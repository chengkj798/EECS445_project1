# Project Intro — Portfolio Lens

Portfolio Lens is a lightweight portfolio assistant built as a Jac-friendly prototype.

A user loads a small portfolio, asks a question in plain English, and gets a short, grounded answer with one helpful visualization.

The current demo can:

- show holdings, prices, and weights
- identify the largest position
- calculate sector exposure
- flag basic concentration risk
- summarize the portfolio in clear language

The prototype is not trying to replace a full investing platform. It is trying to make portfolio questions simpler, faster, and easier to understand.
