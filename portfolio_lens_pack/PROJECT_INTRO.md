# Project Intro — Portfolio Lens

**Portfolio Lens** is a lightweight portfolio assistant built as a Jac-friendly prototype.

The idea is simple: most portfolio platforms give users too much information at once. Portfolio Lens focuses on a smaller and more useful workflow. A user loads a small portfolio, asks a question in plain English, and gets a short, grounded answer with one helpful visualization.

In the current demo, the assistant can:

- show holdings, prices, and weights
- identify the largest position
- calculate sector exposure
- flag basic concentration risk
- summarize the portfolio in clear language

The sample portfolio uses five public U.S. equities across technology, financial services, health care, and consumer goods. That makes the prototype realistic enough for a demo while still keeping the scope small.

What makes this project interesting is not just the UI. Portfolio data has natural structure: holdings connect to sectors, sectors connect to exposures, and user questions ask for a path through that information. That makes the project a strong fit for Jac-style reasoning and graph-oriented logic.

Portfolio Lens is not trying to replace a full investing platform. It is trying to make portfolio questions simpler, faster, and easier to understand.
